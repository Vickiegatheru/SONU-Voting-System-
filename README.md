# SONU Electronic Voting System (SCS3304)

This project is a C-based electronic voting system designed for the **Student Organization of Nairobi University (SONU)**. It is part of the **SCS3304: Programming Assignments** coursework.

## Features
- **Structured Programming**: Developed using modular design and top-down analysis.
- **SONU Governance Structure**: Includes positions for Chairperson, Secretary General, and Treasurers.
- **Persistence**: Election data is stored in binary files within the `data/` directory.

## Current Project Phase
- [x] Assignment 1: Standalone Structured Program.
- [ ] Assignment 2: Connection-oriented Iterative Server.
- [ ] Assignment 3: Distributed Client-Server variations.

## How to Run
1. Compile: `gcc src/main.c -o sonu_portal`
2. Execute: `./sonu_portal`
